# Privacy Policy

Skeleton Replay does not collect, transmit, sell, or share personal data.

The plugin runs inside the user's JetBrains IDE and reads local Skeleton
artifacts selected by the user or generated by the configured local Skeleton
command. It does not send source code, trace artifacts, project metadata, or
usage telemetry to ML Affairs or third-party services.

Any data shown by the plugin remains on the user's machine unless the user
independently chooses to share it.
